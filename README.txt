The java app is located at JFuzzyFinderApp/dist/JFuzzyFinderApp.jar
The source codes are located at JFuzzyFinderApp/src/jfuzzyfinderapp

When the app is opened, select the archive by clicking the select archive button. This automatically loads all the contents of the archive.
Enter the filename in the enter search file or keyword in the search box and click search.

Make sure you click the clear result pane button to avoid search conflicts.
You can also click the open file button when you select the document name in the result pane.
